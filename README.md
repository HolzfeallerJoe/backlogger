# Backlogger

A web interface for tracking your backlog.  
Inspired by the Backlog project from [DarylTalksGames](https://www.youtube.com/@DarylTalksGames).

**Backend**
- FastAPI
- Psycopg

**Database**
- PostgreSQL

**Frontend**
- Jinja2
- Tailwind (via CDN)

You can use Docker Compose to run it locally.
